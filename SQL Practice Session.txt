1. Define data and explain the different types of data.
 Data is defined as raw facts, figures, symbols or observations that can be recorded, stored, and processed to create meaningful information.

Types of Data
1.Quantitative(Numerical)data: Data that can be measured, counted and expressed numerically(Numbers).

2.Qualitative data(Categorical): Data that is not numerical often representing charecteristics or attributes(Text Values).

3.Boolean Data: True/False values

5.Binary Data: Images, files,Videos.

12. Explain the different types of SQL command languages
 1.DDL-Data Definition language
  Used to define database structure.
  Create, Alter, Drop, Truncate.
 2.DML-Data Manipulate Language 
   Used to manipulate data.
   Insert, Update, Delete
 3.DQL-Data Query Language
  Used to retrive data - SELECT
 4.TCL- Transaction Control Language
  Used for transactions. - COMMIT, ROLLBACK, SAVEPOINT

Update:

21. Differentiate between DROP and TRUNCATE commands.?

Delete entire table, Table structure removed, Cannot rollback mostly.
Truncate:
Delete only data, Structure remains, faser opeation


22. Write a query to delete rows where salary is less than the average salary of its department.? 
DELETE FROM employee_details
WHERE salary < (
    SELECT AVG(salary)
    FROM employee_details
);
